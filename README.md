# Lectorium 

**O Workspace Acadêmico Definitivo (PWA)**

O Lectorium é uma plataforma de produtividade acadêmica de alta performance projetada para transformar a maneira como estudantes e pesquisadores interagem com o conhecimento. Operando sob uma arquitetura **Local-First**, a aplicação oferece uma experiência nativa diretamente no navegador, garantindo privacidade, velocidade e disponibilidade total, mesmo sem conexão com a internet.

## 🚀 Funcionalidades Principais

### 1. Leitura Ativa e Inteligente
Um leitor de PDF de classe mundial que vai além da visualização.
*   **OCR Integrado:** Converta imagens e documentos digitalizados em texto selecionável diretamente no navegador.
*   **Anotações Avançadas:** Destaques, notas e desenhos à mão livre que são salvos nos metadados do arquivo, garantindo portabilidade.
*   **Modo de Leitura Imersivo:** Filtros de cor para conforto visual e detecção automática de colunas para artigos científicos.

### 2. Editor de Texto Acadêmico
Escreva artigos e teses sem lutar contra a formatação.
*   **Compatibilidade DOCX:** Edite e exporte arquivos compatíveis com Microsoft Word.
*   **Foco na ABNT:** Ferramentas nativas para citações, referências bibliográficas e formatação acadêmica.
*   **Assistente de Escrita:** Verificação ortográfica e sugestões de estilo em tempo real.

### 3. Organização Visual
*   **Mapas Mentais:** Conecte conceitos e estruture projetos em um canvas infinito.
*   **Gestão de Arquivos:** Sistema de arquivos unificado que combina o armazenamento local do dispositivo com a nuvem.

### 4. Arquitetura Híbrida (Sync Engine)
*   **Offline-First:** Trabalhe em qualquer lugar. Seus arquivos recentes e fixados são mantidos no dispositivo.
*   **Google Drive Nativo:** Sincronização inteligente e transparente com sua conta Google quando online.

### 5. Inteligência Artificial
*   **Assistente de Pesquisa:** Integração com LLMs (Google Gemini) para explicar conceitos complexos, resumir passagens e expandir tópicos de estudo dentro do contexto do seu documento.

## 🛠️ Requisitos de Instalação

Para executar este projeto localmente: 

1.  **Dependências:**
    ```bash
    npm install
    ```

2.  **Variáveis de Ambiente:**
    Crie um arquivo `.env` na raiz do projeto contendo sua chave de API para os serviços de IA:
    ```env
    API_KEY=sua_chave_de_api_aqui
    ```

3.  **Execução:**
    ```bash
    npm run dev
    ```
